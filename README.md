# 🚀 LLD Practices in Python

Welcome to my **Low-Level Design (LLD) practice repository**! This repository serves as a personal laboratory for practicing Object-Oriented Programming (OOP) concepts, SOLID principles, and various Design Patterns using Python.

The goal is to move from theoretical knowledge to practical, clean code implementation.

---

## 📚 Topics Covered

### 1. Object-Oriented Programming (OOP)
*   **Encapsulation:** Data hiding and bundling data/methods.
*   **Inheritance:** Reusability and hierarchy.
*   **Polymorphism:** Method overriding and overloading.
*   **Abstraction:** Using `abc` module (Abstract Base Classes).
